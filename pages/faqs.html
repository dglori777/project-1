Question Answer
What is this site? This site will be used to purchase and view train tickets as well as view train schedules.
How was this site developed? This site was developed in VS Code using a combination of HTML tables, CSS Style Sheets, and Bootstrap classes.
Who is this site used by? Primarily the developer and the instructor to view the work.
What is the content on this site? There is a home page, a page to sign up, a page to log in, and this FAQ page. All pages are accessible using a navbar that is consistent on each page.
What is the purpose of the FAQs page? To fill up space on an extra page.
When will this site become functional? The site is expected to be finished in 2-3 business days.
